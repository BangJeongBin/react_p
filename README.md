## 학습 목표

- 파이썬 기초 문법과 구조를 이해하고 코드를 읽을 수 있게 됩니다
- AI 코드 생성 도구 Cursor를 활용해 간단한 프로그램을 직접 만드는 방법을 배울 수 있습니다.

## 주요 학습 내용

- 이 강의는 입문자들이 가장 많이 선택하는 언어인 파이썬(Python)을 통해 프로그래밍의 기초를 다지고, AI 툴인 Cursor를 활용해서 나만의 프로그램을 작성하는 첫 걸음을 내딛도록 돕는 입문 강의입니다.
- "프로그래밍이 처음이어도 문제없다"는 전제로, 기초 문법부터 실습까지 단계별로 구성되어있습니다. AI가 생성한 코드를 읽을 수 있는 기초를 다지는 것을 목표로 합니다.
- 복잡한 기술적 설명은 피하고, 파이썬 코드를 작동시키기 위해 필요한 것만 쉽고 간단하게 전달합니다.
- AI 코딩 도구 Cursor를 활용하여 코드를 작성하는 훈련을 함께 합니다.
